How I made My Portfolio

So to make my Portfolio I started off creating three boxes, TopBar, MiddleBar and a FooterBar. These bars are relative to the page size, meaning if i change the size of the browser it will relatively change the size of the bars too. You can try the example Here.

To do this i created a container and 3 divs inside that container, I then created class names for each of them respectfully. 

...
<body> 
    <div class="BarContainer">
        <div class="TopBar border">
            <h1>Top Bar</h1>
        </div>
        <div class="MiddleBar border">
            <h1>Middle Bar</h1>
        </div>
        <div class="FooterBar border">
            <h1>Footer Bar</h1>
        </div>
    </div>
</body>
...

I used a more modern approach with the css, using a display type called flex, instead of grid I feel that this approach is more flexable
